# Dotfiles

You have to ask yourself, if you don't have a dotfiles repo, are you a real developer?

The answer is no.

## Requirements

* Make
* Git
* Python
* Bash
* Mutt
* Solarized theme for iTerm2 https://ethanschoonover.com/solarized/
